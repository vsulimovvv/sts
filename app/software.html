Программное
обеспечение

ПО Wialon

Платформа для GPS/ГЛОНАСС мониторинга и IoT, которая позволяет пользователям отслеживать мобильные и стационарные объекты в 130+ странах мира. Разрабатывается компанией Gurtam и распространяется через сеть интеграторов систем спутникового мониторинга.

ПО СКАУТ

ГК СКАУТ российская компания разработчик системы спутникового мониторинга транспорта и производитель GPS/ГЛОНАСС оборудования. В линейке оборудования есть блоки мониторинга и датчики уровня топлива. ПО рассчитано на работу как на собственной линейке оборудования, так и по открытому протоколу со сторонним оборудованием и датчиками.

ПО FORT – MONITOR

Облачная программная платформа для спутникового ГЛОНАСС/GPS мониторинга транспорта. Решение полностью web-ориентировано, не требует установки программного обеспечения на компьютер пользователя. Производится как ПО так и оборудования для мониторинга. Производит оборудование для построения сетей IP-видеонаблюдения, оборудование ЭРА-ГЛОНАСС.

ПО OMNICOMM

Компания Omnicomm — разработчик и производитель системы мониторинга транспорта Omnicomm на базе технологий ГЛОНАСС/GPS, датчиков уровня топлива Omnicomm LLS и дополнительного оборудования. Компания работает как в России, так и во многих странах мира.

ПО АСМ ЭРА

Это коммерческое ПО системы компании АО «ГЛОНАСС». Согласно ПП №2216 данные о ТС категории М2, М3 и перевозящих опасные грузы должны передаваться в систему ЭРА ГЛОНАСС. По заявке владельца ТС он может использовать ПО ЭРА для просмотра данных о ТС в систему ЭРА ГЛОНАСС, так и подключать дополнительные ТС. ПО полностью Российский продукт, который развивается быстрыми темпами.
