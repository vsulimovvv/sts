ПО Wialon

Возможно контролировать ТС, спецтехнику, стационарные объекты, людей – можно выбрать наиболее подходящее под задачу оборудование. Подключить дополнительные датчики, специализированные оборудование, управлять процессами и т.д.

Всегда быть в курсе, где находится Ваша техника, как работает оборудование, получать предупреждения о срабатывании датчиков и много другое, получать информацию на стационарный компьютер или на мобильный телефон или планшет с помощью предложения для мобильных устройств.

В основной версии продукта доступно более 20 специализированных бесплатных веб-приложений, которые позволяют автоматизировать работу служб, в автоматическом режиме контролировать перевозки грузов, работу, простои техники, техобслуживание, решать задачи связанных с логистикой, расписанием пассажирского транспорта, безопасностью движения и многое другое

ПО СКАУТ

ГК СКАУТ российская компания разработчик системы спутникового мониторинга транспорта и производитель GPS/ГЛОНАСС оборудования. В линейке оборудования есть блоки мониторинга и датчики уровня топлива. ПО рассчитано на работу как на собственной линейке оборудования, так и по открытому протоколу со сторонним оборудованием и датчиками.

Использует облачную и локальную версию продукта, предлагает несколько вариантов программ для мониторинга объектов.

  • – ПО СКАУТ Студио, рабочая программа диспетчера. Предназначена для упрощения работы диспетчеров транспортных предприятий, сокращения расходов на содержание транспорта и повышения эффективности работы предприятий. Предназначена для персональных компьютеров.
  • -СильверСтудио – один из пользовательских интерфейсов СКАУТ-Платформы. Для работы СильверСтудио может использоваться любой браузер с установленным плагином Silverlight.
  • - WEB-интерфейс. ПО предназначена для оперативного доступа к данным системы мониторинга с любого устройства. Не требует никаких дополнительных установок и компонентов.

Выпускает собственную линейку блоков мониторинга, с 2021г. это МТ-850/860 с функцией DRV видео контроля и датчиков уровня топлива PetrolX. Поддерживает большое количество протоколов для подключения стороннего оборудования на программную платформу

Компания проводит регулярное обучение клиентов и партнеров с выдачей сертификатов. Регулярно обновляются инструкции по ПО и добавляются новые видеокурсы по работе с программным обеспечением и возможностями оборудования.

ПО FORT – MONITOR

Облачная программная платформа для спутникового ГЛОНАСС/GPS мониторинга транспорта. Решение полностью web-ориентировано, не требует установки программного обеспечения на компьютер пользователя. Производится как ПО так и оборудования для мониторинга. Производит оборудование для построения сетей IP-видеонаблюдения, оборудование ЭРА-ГЛОНАСС.

  • • мониторинг транспорта "в 2 клика";
  • • поддержка терминалов сторонних производителей;
  • • конструктор отчетов;
  • • высокая скорость работы и построения отчетов;
  • • настраиваемая система счетов пользователя;
  • • API для разработчиков.

Выпускает собственную линейку блоков мониторинга FORT. В зависимости от модели помимо стандартных функций в терминалах есть функционал: беспроводное подключение датчиков по BLE, удаленная тарировка ДУТ, встроенный CAN-сканнер, дополнительный Wi-Fi модуль, а также оборудование аппаратуры спутниковой навигации (АСН) и устройства вызова экстренных оперативных служб (УВЭОС).

Поддерживает большое количество протоколов для подключения стороннего оборудования на собственную программную платформу.

  • • Функционал Fprt Monitor постоянно пополняется новыми решениями, что позволяет пользователям получать новые возможности расширения контроля за автопарком.
  • • Разработан инструмент для самостоятельного создания отчетов под задачи, которые необходимо решать
  • • Осуществлена поддержка основных терминалов, которые присутствуют на рынке

ПО OMNICOMM

Компания Omnicomm — разработчик и производитель системы мониторинга транспорта Omnicomm на базе технологий ГЛОНАСС/GPS, датчиков уровня топлива Omnicomm LLS и дополнительного оборудования. Компания работает как в России, так и во многих странах мира.

Использование неоднократно проверенных стандартных решений для различных отраслей транспорта позволяет точно прогнозировать результат от внедрения системы мониторинга

Полный перечень базового функционала, внедрение инструментов видеоконтроля, анализ работы водителей, отчеты по геозонам, интеграция в существующие ERP системы и получение уведомлений о нарушениях позволяет значительно повысить автотранспортную безопасность на предприятии

Пожизненная гарантия, Менее 0,3% гарантийных случаев, Лидер по объемам установок в СНГ, Полный набор сертификатов (EAC, СИ, CE, E -mark, Ex, FCC, Anatel)

Простота, доступность и интуитивная понимание возможностей системы позволяет очень быстро освоить ее любому пользователю

Более чем 20 летний опыт работы на рынке и множество реализованных проектов
