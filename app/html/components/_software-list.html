ПО АСМ ЭРА

Это коммерческое ПО системы компании АО «ГЛОНАСС». Согласно ПП №2216 данные о ТС категории М2, М3 и перевозящих опасные грузы должны передаваться в систему ЭРА ГЛОНАСС. По заявке владельца ТС он может использовать ПО ЭРА для просмотра данных о ТС в систему ЭРА ГЛОНАСС, так и подключать дополнительные ТС.
